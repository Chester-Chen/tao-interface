## [后台管理系统及移动端的数据api服务](https://github.com/Chester-Chen/tao-interface)

基于 nodejs 开发

技术栈: nodejs + koa2 + koa-bodyparse + koa-router + koa2-cors + mongoose


# 移动端build && setup

```bash

# 安装依赖
`npm i`

# 本地调试
`npm run serve`

# 编译
`npm run build`

```
